学习Python时候写的一些代码，内容如下……
wechat_server.py：微信公众号的后台，有2个功能：1)接收用户发送的消息，倒序回复给用户；2)如果用户发送的内容包含‘笑话’，则回复一条笑话给用户。笑话是从一个笑话网站上爬的，每10分钟更新一次，每次存储20条。
xDy.py：帮产品算的。掷x次，有y个面的骰子，得到各个结果的概率分别是多少。例子里是投掷6次4面骰子。
xAyB.py：猜不重复的4位数字。如果位置和数字都对，记1个A，如果数字对位置错，记1个B。每次猜数字都会返回A,B的数量，猜对了就恭喜你~
picture.py：通过运行参数，选取图片，字符画保存位置，宽高。将选择的图片，转换成一个字符画
dichotomie.py：中值法找一个数字在一个数组中的位置
fibs.py：每个数是前2个数之和。
TurningStatement.py：输入的内容按照空格分割倒序输出
appium：appium，启动计算器进行一些操作；打开一个web获取某个控件，检查控件名称，输入操作
selenium：selenium使用chrome浏览器打开百度首页，通过xpath找到登录按钮并点击，最后关闭浏览器
dataTrans：把程序拔下来的数据，转换成老板能看的懂的格式
TM：c/s架构的聊天室。服务器会把每个客户端发来的消息群发给每个连接中的客户端。客户端没有好好整，丑见谅
web+sql：数据库里预存了一些用户的喜好数据，在web端输入一个用户的ID，可以根据用户的喜好推荐一个他最近没有使用的同类新品
unittest：单元测试框架
ichat：让自己的微信变成聊天机器人，调用的图灵机器人的接口，自动回复聊天消息
